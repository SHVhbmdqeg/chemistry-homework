# chemistry-homework
化学寒假作业

# 核心功能
1. 周期表展示
- 标准7行18列布局，包含118种元素
- 镧系锕系单独展示在底部
- 按元素类型着色
- 元素卡片显示原子序数、符号、名称、质量
2. 双图层切换
- 元素类型颜色图层 ：按元素类型分类着色
- 原子半径图层 ：通过圆圈大小和颜色渐变展示原子半径差异
- 图层状态自动保存，返回时保持
3. 搜索与筛选
- 支持按名称、符号、原子序数搜索
- 按类别筛选（金属、非金属、类金属、卤素、稀有气体）
- 显示全部/隐藏标签功能
- 点击行列标签筛选对应元素
4. 元素详情页
- 基本信息：原子序数、质量、分类、电子排布、电负性、原子半径、密度、熔沸点、发现年份、发现者
- 详细信息：基本性质、原子结构、主要用途、制备方法
- 动态原子结构图：可视化展示原子核和电子层分布
5. 响应式设计
- 区分手机端电脑端
